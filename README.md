# 🌟 Project Overview

**QuickEats** is a modern and scalable **Food Delivery Application** designed to provide users with a smooth and fast online food ordering experience.

The platform combines:

✅ Beautiful modern UI  
✅ Restaurant & customer dashboards  
✅ Secure authentication system  
✅ Real-time order management  
✅ Online payment integration  
✅ Responsive mobile-friendly design  
✅ Fast React frontend with Spring Boot backend  

---

# ✨ Core Features

## 🔐 Authentication & Security

- Secure login & signup
- JWT Authentication
- Role-based access
- Email verification system

### User Roles

- 👤 Customer
- 🍽 Restaurant Admin
- 🛵 Delivery Partner

---

# 🍔 Customer Features

Customers can:

✔ Browse restaurants  
✔ Search food items  
✔ Filter by category  
✔ Add items to cart  
✔ Place online orders  
✔ Track live order status  
✔ View order history

Restaurant admins can:

# ⚙️ Installation & Setup

# 🟦 Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# 🟩 Backend Setup

```bash
cd backend

mvn clean install

mvn spring-boot:run
```

---

# 🗄️ MySQL Configuration

Update your `application.properties` file:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/quickeats
spring.datasource.username=root
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
```
